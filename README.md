## Etapes
- Executer `rails db:seed` pour créer des catégories et des articles
- Executer `rails server`
- Executer `rails test`

## Technologies
#### Frontend
- jQuery
- Foundation (style, modale et carrousel)
- Slim
- Froala WYSIWYG editor (contenu formattable)
- Kaminari

#### Backend
- Rails 5.0.1
- Ruby 2.3.3

### Base de données
- SQlite 3

#### Tests
- Shoulda
